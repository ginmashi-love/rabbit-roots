# rabbit-roots
里親うさぎたちのルーツと広がりを、みんなでそっと見守る場所。

このサイトは、
個人が自分のうさぎの家系図を作り、
血のつながりがあると分かったときだけ、
お互いの同意のもとで静かにつながっていく
そんな場所を目指しています。

SNSではありません。
正しさを主張する場所でもありません。

時間の中で広がっていく
「命のつながり」を、
あとから振り返れるように残すための
公開できる、個人的な図鑑です。

このサイトでできること

自分のうさぎのプロフィールを登録できます
（名前、誕生日、写真、好きな食べ物、好きなこと など）

父・母・兄弟姉妹・子孫などの家系を、自分の範囲で整理できます

うさぎのページは、誰でも閲覧可能です

血縁が分かった場合、コード交換＋相互承認によって
家系図のつながりを確定できます

昔飼っていた子の子孫が、
どこかで生きていることを知ることができます

このサイトが大切にしている考え方
血縁リンクは「相互同意」がすべてです

血縁の線は、両者が同意したときだけ表示されます

同意がない関係は、推測も含めて表示されません

一度つながった関係も、オーナーは解除できます

このサイトでは
「合意された関係」＝「確定した関係」
として扱います。

証拠の提出や、第三者による判定は行いません。

正しさを主張しない

血統の正確さを評価したり、ランク付けしたりしません

議論や投票で真偽を決めることもしません

誰かを否定するための場所にはしません

ここにある関係は、
当事者同士が信じている記憶です。

人より、うさぎが主役

飼い主の本名、住所、連絡先は扱いません

フォロー、DM、タイムラインはありません

飼い主同士が無理につながる必要はありません

血のつながりをたどった結果、
「人がゆるくつながる」ことがあってもいい。
でも、それは副産物です。

血縁リンクの仕組み（コード交換型）

公開されているうさぎページを見て
「もしかして血縁かも？」と気づく

相手から リンク用コード を教えてもらう
（サイト外でのやり取り）

自分の管理画面からコードを入力して申請

相手が承認したとき、はじめて家系図につながる

承認されるまで、公開ページには何も変化はありません。

このサイトで「しないこと」

血統の真偽判定

証拠の提出・審査

SNS的な交流（いいね・DM・フォロー）

個人を特定する情報の収集

無断での血縁登録

これらは、あえてやりません。

なぜこのサイトを作るのか

里親募集で出会ったうさぎたちには、
その前の時間があり、
その先の時間があります。

「おばあちゃんが同じだった」
「昔飼っていた子の血が、こんなに広がっていた」

そんな気づきが、
誰かの記憶と誰かの現在を
やさしくつなげてくれると信じています。
